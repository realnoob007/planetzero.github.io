- **gpt-3.5-turbo** - 响应最快的模型，适合处理日常事务

- **gemini-pro** - 谷歌最新模型
- **llama-13b-chat-v2** - 很有意思的开源模型
- **智谱AI** - 中国 · 智谱AI(清华大学)
- **文心一言** - 国产大模型，百度最新一代产品

### 高级模型：
- **gpt-4** - 公认性能最强模型，逻辑理解能力很强
- **gpt-4全能版** - 在gpt-4的基础上内置了联网和画图的能力
- **gpt-4-32k** - 将gpt-4的上下文长度扩大到32k
- **claude-2** - Anthropic最强模型，上下文长度扩大到100k
  
### 绘画模型：
- **Midjourney** - 目前性能最强模型
